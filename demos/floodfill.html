How It Works

The C source compiles to 0 ISA instructions. The flood fill uses a stack-based iterative approach: pop a cell, color it, push uncolored neighbors. Each cell fill takes ~8 ISA instructions. Walls (black cells) block the fill. Click any non-wall cell to start filling.

ISA Interpreter executes instantly. ONNX Transformer runs each instruction as a 146x512 matrix multiply via WebGPU. You can watch the fill spread cell by cell through the neural network.
