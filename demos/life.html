How It Works

The Game of Life rules are written in C, compiled to 570 ISA instructions, and executed on a scaled-up Loom computer with a 164×2048 state tensor. The standard 155×1024 model supports 928 instruction slots; this program requires the larger variant. The same 8-layer, 22-opcode ISA is used. Only the weight matrix dimensions change.

The 4×4 grid uses two 16-element arrays, current and next, in the transformer's 64-slot memory. Click cells to toggle them before running.
